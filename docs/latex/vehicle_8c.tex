\doxysubsection{Detailed Description}
File containing the vehicle functions. 

This file contains the implementation of functions related to the management of vehicles. Functions include reading vehicles from a text file, creating a vehicle, creating a vehicle list, printing a vehicle list, editing a vehicle, deleting a vehicle, storing a vehicle list in a binary file, searching for a vehicle by registration, checking if a vehicle is available, and editing a vehicle\textquotesingle{}s availability.
