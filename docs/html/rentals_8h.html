Creates a new rent.

This function creates a new rent with the given vehicle registration, user NIF, and time in minutes. It also checks if the user and vehicle exist and if the vehicle is available. If any of these conditions are not met, the function returns NULL. Otherwise, it creates a new rent and returns it.
